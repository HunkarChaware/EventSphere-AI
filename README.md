# EventSphere AI 🎪 — Smart Physical Event Experience Platform

A production-ready full-stack web application that enhances real-world physical event experiences through AI-powered assistance, live dashboards, smart recommendations, and intelligent queue management.

## 🎯 Features

### ✨ Core Features

1. **AI-Powered Chatbot Assistant**
   - Intelligent event-related query answering
   - Contextual understanding using Google Gemini API
   - Real-time event information integration
   - Graceful fallback responses

2. **Live Event Dashboard**
   - Real-time event updates and status tracking
   - Interactive event cards with capacity visualization
   - Filter and search functionality
   - Event recommendations based on user interests

3. **Smart Recommendation Engine**
   - Personalized event suggestions
   - Interest-based filtering
   - Trending event highlights

4. **Queue & Crowd Management**
   - Token-based queue system
   - Estimated waiting time calculation
   - Real-time capacity indicators
   - Queue position tracking

5. **Admin Panel**
   - Event creation and management
   - Schedule updates
   - Analytics and statistics
   - User management

## 🛠️ Tech Stack

### Frontend
- **React 18** - UI library with Vite for fast development
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing
- **Axios** - HTTP client
- **Lucide React** - Icon library

### Backend
- **Node.js** - JavaScript runtime
- **Express** - Web framework
- **MongoDB** - NoSQL database
- **Mongoose** - ODM for MongoDB
- **Google Gemini API** - AI chatbot integration

### Deployment
- **Frontend**: Vercel
- **Backend**: Render or Railway
- **Database**: MongoDB Atlas
